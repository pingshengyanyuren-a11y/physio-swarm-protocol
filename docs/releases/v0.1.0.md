# Physio Swarm Protocol v0.1.0

Initial public release of the physiological swarm framework.

## What is included

- protocol primitives for homeostasis, task signals, and cell state
- an explicit signal bus for endocrine and immune traffic
- organ systems for endocrine, nervous, metabolic, and immune control
- a runtime that routes work through those organs
- a runnable research assistant demo
- a bundled Codex skill for physiological swarm design

## What this is

- a protocol-first framework seed
- a standalone repository
- a runnable reference implementation

## What this is not

- not a production-ready agent platform
- not a benchmark suite
- not another role-playing org-chart swarm
