# v0.1.6

## Added

- `ContrastiveLatentModel` for stronger positive/negative online latent learning
- `integrate()` support in `SemanticVectorBus` for finer continuous field updates
- `NetworkTissueExecutor` for remote node execution over HTTP

## Changed

- runtime now defaults to stronger contrastive latent learning
- latent fields can be integrated over duration and timestep instead of single coarse ticks
- tissue execution can target both local worker processes and remote node endpoints
