# Physio Swarm Protocol v0.1.1

Second iteration of the physiological swarm framework.

## What improved

- added an organ registry for building cells from organ specs
- added reserve-cell promotion when a primary cell is quarantined
- added a recovery path that can restore quarantined cells
- added a lightweight workflow DSL for stage-based execution
- tightened the public documentation so the repository describes what it actually does now

## Why this matters

The repository is now meaningfully closer to a reusable framework and not only a protocol sketch.
